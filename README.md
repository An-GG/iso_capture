# iso_capture
Fetch Disk Captures and Images from Platforms

## Definition

**Input**
UID + Platform ID

**Action**
Fetch All (associated?) Image and ISO Content

## Notes

- filter by
    - is retweet
    - is reply
    - is NPC or PC (non-playable) or (playable)
- options
    - get metadata or do not get metadata
    - output ordering for namespace

## Support
All major platforms.
